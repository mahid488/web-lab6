# Spiritual Counter App

A simple web application for spiritual counting, such as tasbih or prayer beads.

## Features

- Counts from 1 to customizable maximum per cycle
- Tracks completed cycles
- Sets spiritual goals based on cycles
- Saves progress in localStorage
- Responsive design with gradient background

## Usage

Open `index.html` in a web browser to start using the app.

## Files

- `index.html`: Main HTML file
- `styles.css`: CSS styles
- `script.js`: JavaScript functionality
